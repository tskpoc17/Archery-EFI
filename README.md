# Archery-EFI
Equipment Forgiveness Index<br>
採用箭群平均中心為分離變因標準<br>
依變因箭數校正計算個別MAD<br>
精準度會隨距離拉遠而放大<br>
故增加距離精準度修正係數K<br>
以方便推估不同距離下之EFI<br>
公式不需倚賴機械放箭器<br>
將更符合射手實際應用<br>
對以有限距離推估更遠距離計算亦適用<br>
此公式不僅能提供射手參考比較<br>
各配置間之精準度寬容性<br>
亦可運用於檢視修改自身動作成效<br>
1. 請輸入 12 支箭的 X 軸座標<br>
  （距靶心公分數 靶心右側為-<br>
    請寫至小數點以下一位）<br>
   座標之間請用「半形逗點 , 」隔開<br>
3. 設定K 值（預設為 1）<br>
   K=欲推估距離/實測距離<br>
4. 點擊按鈕算出 EFI 器材寬容性指標</div>
